📚 Smart Library Management System

A modern digital library platform designed to manage books, users, subscriptions, and intelligent services with automation and ease.

🚀 Key Features

📖 Book Management

Admin can add, update, and remove books. Users can browse available books, view details, and check availability in real-time.

👤 User Management

Users can register, log in, manage profiles, and track their issued books, reading progress, and history.

🔍 Smart Book Search & Filter

Easily find books using filters like genre, title, or author. Helps users quickly discover relevant books.

❤️ Wishlist

Users can save books to their wishlist and access them later for reading or issuing.

🤖 AI Chatbot Assistant

Users can interact with an AI chatbot for book-related queries, guidance, and mental wellness support. This feature is completely free.

⭐ Review & Rating System

Users can read reviews of books. To submit a review, the user must first issue and read the book.

🔐 System Rules & Validations

  • ✔ Active subscription is required to issue books
  • ✔ Without subscription, issuing is restricted
  • ✔ AI chatbot is free for all users
  • ✔ Reviews allowed only after reading the book
  • ✔ Book must be available before issuing
  • ✔ Return must be approved by admin

💳 Subscription & Payments

Users must purchase a subscription to unlock premium features like book issuing. The system provides secure and fast payment processing.

  • 📅 Flexible plans (Monthly / Yearly)
  • 🔐 Secure payment gateway
  • ⚡ Instant activation

⚠️ Fine & Penalty System

Our system uses a dynamic and priority-based fine calculation model based on book condition, delay duration, and subscription status.

  • 📕 Book Loss (Highest Priority):
    • Active Subscription → 5 × Book Price
    • Expired Subscription → 10 × Book Price
  • 📖 Book Damage:
    • Active → 1/3 of Book Price
    • Expired → 2/3 of Book Price
  • ⏰ Overdue Fine:
    • Calculated every 3 days (block system)
    • Active → ₹5 per block
    • Expired → ₹10 per block
  • ⚙️ Processing Fee: ₹20 (optional)

🔴 If a book is marked as lost, no other fines are applied.
🟢 Damage and overdue fines can be combined.
⚠️ All fines must be cleared before issuing new books.

📊 User Progress & Tracking

Users can track issued books, overdue status, fines, and reading history through a personalized dashboard.

This system combines automation, intelligent services, and a user-friendly interface to deliver a seamless and modern library experience.
